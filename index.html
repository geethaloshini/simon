SIMON GAME

Test your memory and see how far you can go!

Good luck, and have fun challenging your memory with Simon!

Tips for success

  • Stay Focused: Pay close attention to each sequence. It helps to mentally rehearse each step before pressing the buttons.
  • Practice Makes Perfect: The more you play, the better your memory will become.

Rules for the game

  1. Start the Game:
    • Press the ENTER button to begin. SIMON will give you the first signal
  2. Repeat the Signal:
    • Once SIMON gives a signal, replicate it by pressing the corresponding button.
  3. Sequence Expansion:
    • After you correctly repeat the signal, SIMON will add a new signal to the sequence.
    • You will now need to repeat the entire sequence, including the new addition.
  4. Keep Going:
    • SIMON will continue to add one new signal to the sequence each time you successfully replicate it.
    • Your task is to remember and repeat the increasingly longer sequence.
  5. Winning the Game:
    • The game continues as long as you can correctly repeat the sequence.
    • The maximum level to win is 10. Can you reach it?
